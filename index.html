Sobre este curso

Este libro web ha sido diseñado con estándares editoriales universitarios para la enseñanza de ROS 2 (Robot Operating System 2) desde cero hasta nivel avanzado. Abarca desde la instalación del entorno de desarrollo hasta la simulación en Gazebo Harmonic, pasando por la arquitectura de nodos, tópicos, servicios, URDF, SLAM y navegación autónoma.

El contenido está construido sobre tecnologías web modernas (HTML5 semántico, CSS3, JavaScript ES6) y se publica de forma gratuita a través de GitHub Pages, garantizando acceso universal y actualizaciones continuas.
